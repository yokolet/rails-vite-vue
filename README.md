# Vite + Vue + Bun on Rails

This is a sample Ruby on Rails application whose frontend is created using
Vite, Vue ans Bun.

Details are in the blog post, https://yokolet.com/blog/

### Versions

- Ruby 3.2.3
- Rails 7.1.3.2
- Node.js v21.5.0
- Bun 1.0.29
- Vite 5.1.4
- Vue 3.4.21

### How to Run the App

1. bundle install
2. bun install
3. bin/dev
4. Open http://localhost:3000/
